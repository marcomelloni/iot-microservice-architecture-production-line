IoT Microservice Architecture for a Production Line composed of multiple Robotic Arms

Overview

This web interface is part of the IoT microservice architecture production line. It serves as the Web UI microservice, providing a Flask-based interface to monitor telemetry data from robotic systems in real-time.

The Web UI fetches data from an HTTP API and visualizes important metrics such as joint consumption and weight capacity of robotic systems.

Features

  • Fetch and display real-time joint consumptions of a robotic arm data
  • Fetch and display real-time weight supported by the end effector of a robotic arm

Available URLs

The following API endpoints are available in this service:

  • /robot/<robot_id>/telemetry/joints_consumption: Fetches and displays joint consumption data for a specific robot.
  • /robot/<robot_id>/telemetry/weight_ee: Fetches and displays weight capacity data for a specific robot.
